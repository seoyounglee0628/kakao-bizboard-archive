# 카카오 비즈보드 아카이브

카카오톡 비즈보드 광고 스크린샷을 브랜드·매체·소재 사이즈·타깃(연령/성별)·날짜별로 모아 정리하고 검색하는 개인용 웹앱입니다.

## 접속 방법

**https://seoyounglee0628.github.io/kakao-bizboard-archive/** 를 북마크해두고 바로 열면 됩니다. 서버를 켜거나 터미널을 쓸 필요 없습니다.

(로컬에서 오프라인으로 쓰고 싶으면 `./start.sh`로 로컬 서버를 띄워 http://localhost:8787 로 접속할 수도 있습니다. 단, 브라우저마다 저장 공간이 분리되어 있어서 배포된 사이트와 로컬 서버는 데이터가 서로 다르게 쌓입니다 — 한쪽으로 통일해서 쓰는 걸 권장합니다.)

## 데이터 수집 방법 (중요)

이 프로그램은 광고를 자동으로 수집하지 않습니다. 카카오는 비즈보드 광고 아카이브 공개 API를 제공하지 않고, 자동으로 여러 계정을 돌려 타깃별 노출 광고를 긁는 방식은 카카오 이용약관 위반 소지가 있습니다.

대신 이렇게 사용하세요:
1. 실제 기기(본인 폰, 테스트 기기 등)에서 카카오톡을 볼 때 노출된 비즈보드 광고를 스크린샷으로 캡처
2. 이 앱에서 "+ 광고 추가"로 업로드 → 필요하면 잘라내기(크롭)로 불필요한 여백 제거
3. 브랜드/카피 문구는 자동 인식되어 채워짐 (아래 참고), 확인 후 필요하면 수정
4. 소재 사이즈(가로·세로·정방형)/매체(Google·Meta·Kakao·CPA·TMS 중 선택)/소재 형식/타깃 연령·성별/캡처 날짜 등 나머지 항목을 태그
5. 갤러리에서 검색·필터로 모아보기

## 이미지 자르기 (크롭)

이미지를 업로드하면 저장 전에 크롭 화면이 먼저 뜹니다. 원하는 영역으로 자른 뒤 "자르기 적용"을 누르면 그 결과가 저장되고, 자를 필요가 없으면 "원본 그대로 사용"을 누르면 됩니다. 저장 후에도 미리보기 아래 "✂️ 다시 자르기" 버튼으로 다시 자를 수 있습니다. 자르기 이후에 카피/브랜드 자동 인식이 실행되므로, 브랜드명이 있는 부분을 잘라내면 자동 인식이 안 될 수 있습니다.

## 자동 인식 (OCR)

이미지를 업로드하면 브라우저 안에서 Tesseract.js로 텍스트를 인식해서 카피 문구를 채우고, 인식된 텍스트에 알려진 브랜드명이 있으면 브랜드도 자동으로 채웁니다. 인식 전에 이미지를 확대·흑백 대비 강조 처리하고, 배너 광고처럼 텍스트가 흩어진 레이아웃에 맞는 인식 모드를 사용해서 정확도를 높였습니다. 이미지 자체는 외부로 전송되지 않고 브라우저 안에서만 처리됩니다 (단, OCR 엔진 파일은 최초 1회 CDN에서 내려받으므로 그때만 인터넷 연결이 필요합니다).

로고만 있고 텍스트가 없거나, 목록에 없는 브랜드는 자동으로 인식되지 않을 수 있습니다 — 자동 채움 결과는 항상 확인 후 필요하면 수정해주세요. `app.js`의 `KNOWN_BRANDS`에 브랜드를 추가하면 인식 가능한 브랜드를 늘릴 수 있습니다.

매체는 정해진 목록(Google, Meta, Kakao, CPA, TMS) 중에서 고르는 드롭다운입니다. 목록을 더 추가하고 싶으면 `index.html`에서 `id="inMedia"`와 `id="fMedia"` 두 군데의 `<option>`을 같이 추가하면 됩니다.

## 데이터 저장 위치

모든 광고 데이터(이미지 포함)는 브라우저의 IndexedDB에 저장되며, 외부 서버로 전송되지 않습니다. 브라우저 데이터를 지우면 사라지므로, 상단의 "내보내기" 버튼으로 주기적으로 JSON 백업을 받아두는 것을 권장합니다. "가져오기"로 백업 파일을 복원할 수 있습니다.

## 파일 구성

- `index.html` — 화면 구조
- `style.css` — 스타일
- `app.js` — 저장/필터/검색/크롭/OCR 자동 인식 로직 (IndexedDB + Cropper.js + Tesseract.js)
- `start.sh` — 로컬 서버 실행 스크립트
