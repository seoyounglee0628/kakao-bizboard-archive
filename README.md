# 카카오 비즈보드 아카이브 (로컬 전용)

카카오톡 비즈보드 광고 스크린샷을 타깃(연령/성별/지역)·업종·브랜드·날짜별로 모아 정리하고 검색하는 개인용 로컬 웹앱입니다.

## 실행 방법

```bash
./start.sh
```

브라우저가 자동으로 열리지 않으면 http://localhost:8787 로 직접 접속하세요.

(`index.html`을 더블클릭해서 바로 열어도 대부분 동작하지만, 브라우저에 따라 저장이 불안정할 수 있어 위 방법을 권장합니다.)

## 데이터 수집 방법 (중요)

이 프로그램은 광고를 자동으로 수집하지 않습니다. 카카오는 비즈보드 광고 아카이브 공개 API를 제공하지 않고, 자동으로 여러 계정을 돌려 타깃별 노출 광고를 긁는 방식은 카카오 이용약관 위반 소지가 있습니다.

대신 이렇게 사용하세요:
1. 실제 기기(본인 폰, 테스트 기기 등)에서 카카오톡을 볼 때 노출된 비즈보드 광고를 스크린샷으로 캡처
2. 이 앱에서 "+ 광고 추가"로 업로드 → 브랜드/업종/카피 문구는 자동 인식되어 채워짐 (아래 참고)
3. 자동 인식된 내용을 확인하고, 타깃 연령·성별/캡처 날짜 등 나머지 항목을 태그
4. 갤러리에서 검색·필터로 모아보기

## 자동 인식 (OCR)

이미지를 업로드하면 브라우저 안에서 Tesseract.js로 텍스트를 인식해서 카피 문구를 채우고, 인식된 텍스트에 알려진 브랜드명이 있으면 브랜드/업종도 자동으로 채웁니다. 이미지 자체는 외부로 전송되지 않고 브라우저 안에서만 처리됩니다 (단, OCR 엔진 파일은 최초 1회 CDN에서 내려받으므로 그때만 인터넷 연결이 필요합니다).

로고만 있고 텍스트가 없거나, 목록에 없는 브랜드는 자동으로 인식되지 않을 수 있습니다 — 자동 채움 결과는 항상 확인 후 필요하면 수정해주세요. `app.js`의 `BRAND_CATEGORY_MAP`에 브랜드를 추가하면 인식 가능한 브랜드를 늘릴 수 있습니다.

## 데이터 저장 위치

모든 광고 데이터(이미지 포함)는 브라우저의 IndexedDB에 저장되며, 외부 서버로 전송되지 않습니다. 브라우저 데이터를 지우면 사라지므로, 상단의 "내보내기" 버튼으로 주기적으로 JSON 백업을 받아두는 것을 권장합니다. "가져오기"로 백업 파일을 복원할 수 있습니다.

## 파일 구성

- `index.html` — 화면 구조
- `style.css` — 스타일
- `app.js` — 저장/필터/검색/OCR 자동 인식 로직 (IndexedDB + Tesseract.js)
- `start.sh` — 로컬 서버 실행 스크립트
